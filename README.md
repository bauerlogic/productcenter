# ProductCenter

A product management application.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

```bash
npm install
```

### Development

Start the server with auto-reload on file changes:

```bash
npm run dev
```

The app runs at **http://localhost:3000**.

### Production

```bash
npm start
```

### Usage

- Open **http://localhost:3000** in your browser
- Select a product YAML file from the sidebar or create a new one
- Edit fields via the **Form View** or switch to **YAML View** for raw editing
- Add/remove parameters with the **+ Add Property** and **✕** buttons
- Save with the **Save** button or **Ctrl+S**

### Product YAML Files

Product definitions are stored as YAML files in the `products/` directory. See `products/ethernet-switch-ES-2408G.yaml` for an example template.
